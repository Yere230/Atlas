# HOME SCREEN

## Objetivo

La pantalla principal debe hacer que el usuario sienta que tiene el control de su dinero en menos de 5 segundos.

No debe parecer una hoja de cálculo.

No debe mostrar demasiados gráficos.

Debe sentirse como abrir una conversación con un asesor financiero.
