# DESIGN SYSTEM

## Objetivo

El Design System define las reglas visuales de Atlas para que toda la aplicación mantenga una identidad consistente.

---

# Filosofía

La interfaz debe transmitir:

- Confianza
- Claridad
- Tranquilidad
- Profesionalismo
- Simplicidad

Nunca debe sentirse saturada de información.

---

# Colores

## Principal

Verde oscuro

Se utiliza para botones principales, elementos importantes y la identidad de Atlas.

## Secundario

Verde esmeralda

Para indicadores positivos y progreso.

## Fondo

Blanco

Debe predominar en toda la aplicación.

## Neutros

Grises claros para separar secciones sin generar ruido visual.

## Acento

Dorado

Solo para destacar logros importantes o funciones premium.

---

# Tipografía

Inter

Debe utilizarse en toda la aplicación.

---

# Bordes

Todos los botones y tarjetas tendrán bordes redondeados.

La sensación debe ser moderna y amigable.

---

# Iconografía

Iconos minimalistas.

Sin exceso de detalles.

---

# Atlas

Atlas siempre estará representado por su símbolo minimalista.

Cada recomendación importante mostrará el ícono de Atlas para que el usuario la identifique rápidamente.

---

# Regla Atlas

Nunca mostrar datos sin explicar:

- Qué significan.
- Por qué son importantes.
- Qué acción debería tomar el usuario.
